Install QoF

Installing QoF is straightforward. Follow the steps below to get up and running.

PlayStation and Xbox do not support direct installation of third-party addons. QoF is only available on Windows, Android, and iOS.

Before You Start

Minecraft Bedrock Edition
QoF is a Bedrock-only addon. Java Edition is not supported.[1]
Beta APIs (Experimental)
Required. Without this toggle enabled, QoF will not function. You'll enable this during the world setup in step 4.

Installation

  1. Download a release
    Head to the changelog and pick a version. When in doubt? grab the latest stable release.
  2. Open the .mcaddon file
    Tap or double-click it. Minecraft will launch and import the addon automatically. You should see a double confirmation toast in-game.
  3. Create or edit a world
    Go to Play -> Create New (or edit an existing world). Under the Add-Ons tab, find QoF and apply it.
  4. Enable Beta APIs
    In world settings, open the Experiments tab and toggle on Beta APIs. This step is required.
  5. Configure modules
    Once inside the world, open pack settings to enable, disable, or tweak individual QoF modules to your liking.

Got stuck? Try Troubleshooting, or open an issue on GitHub.
