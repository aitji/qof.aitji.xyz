Troubleshooting

A list of known issues and how to resolve them. If your problem isn't listed here, open an issue on GitHub or reach out to @aitji directly.

In-Game

QoF doesn't seem to be doing anything in-game
Make sure the Beta APIs is enabled under Experiments in your world settings. This is the most common cause — QoF will warn you in-game and won't work without it. See Installation -> Step 4 for guidance.

Installation

The .mcaddon file didn't open Minecraft
Try opening Minecraft first, then tap the file from your file manager or downloads. On Android you may need to set Minecraft as the default app for .mcaddon files.
I updated QoF but my world still uses the old version
Re-import the new .mcaddon file, then go back into your world's Add-Ons settings, remove the old version, and apply the new one. Existing worlds do not update automatically.

Still stuck? Open an issue on GitHub or contact @aitji.
