Modules

Modules are self-contained features included in the QoF addon. Each module targets a specific gameplay mechanic and is designed to remain consistent with vanilla Minecraft behavior, QoF modules prioritize small, meaningful improvements over major changes, ensuring compatibility with the core gameplay experience.
